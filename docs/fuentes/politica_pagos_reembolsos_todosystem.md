# Política de Pagos y Reembolsos - Instituto TodoSystem

## 1. Propósito

La presente política establece orientaciones generales sobre pagos, confirmación de valores, acuerdos administrativos y solicitudes relacionadas con reembolsos en el Instituto TodoSystem.

Esta información es de referencia para el proyecto académico TodoSystem EduBot y debe ser validada con el área administrativa antes de tomar decisiones finales.

## 2. Confirmación de valores

Los valores de matrícula, mensualidades, materiales o certificados deben ser confirmados directamente con el área administrativa del instituto.

El agente TodoSystem EduBot no debe inventar valores ni entregar precios definitivos si estos no se encuentran registrados en la base de conocimiento.

## 3. Formas de pago

Las formas de pago disponibles deben ser confirmadas con el área administrativa.

Según las políticas vigentes, el instituto puede orientar al estudiante sobre pagos en efectivo, transferencias u otras alternativas, siempre que estas estén oficialmente habilitadas.

## 4. Pagos por cuotas

La posibilidad de pagar por cuotas debe ser consultada directamente con el área administrativa, ya que puede variar según el curso, la duración, la jornada o las condiciones institucionales.

## 5. Reembolsos

Las solicitudes de reembolso deben ser revisadas por el área administrativa.

Para solicitar una revisión, el estudiante debe presentar la información relacionada con el pago realizado, el curso matriculado y el motivo de la solicitud.

La aprobación o rechazo del reembolso dependerá de las políticas internas del instituto.

## 6. Cambios de curso

En caso de que un estudiante desee cambiar de curso, debe consultar con el área administrativa si existe disponibilidad y si el cambio es posible según las condiciones académicas y administrativas.

## 7. Recomendación al estudiante

Antes de realizar cualquier pago, se recomienda confirmar:

- Nombre del curso.
- Valor actualizado.
- Forma de pago autorizada.
- Fecha de inicio.
- Condiciones de devolución o cambio.
- Datos oficiales de contacto.

## 8. Fuente

Documento interno de referencia creado para el proyecto académico TodoSystem EduBot.
