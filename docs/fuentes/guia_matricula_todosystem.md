# Guía de Matrícula - Instituto TodoSystem

## 1. Presentación

El Instituto TodoSystem ofrece programas de formación práctica orientados al desarrollo de competencias en tecnología, informática, herramientas digitales y áreas administrativas.

Esta guía presenta información general sobre el proceso de matrícula para estudiantes interesados en participar en los cursos ofrecidos por el instituto.

## 2. Proceso de matrícula

El proceso de matrícula está compuesto por las siguientes etapas:

1. Consultar la oferta de cursos disponible.
2. Seleccionar el curso de interés.
3. Confirmar disponibilidad de cupo con el área administrativa.
4. Diligenciar el formulario de inscripción.
5. Presentar el documento de identidad.
6. Confirmar los requisitos específicos del curso.
7. Realizar el proceso de pago según las indicaciones del área administrativa.

## 3. Documentos requeridos

Para realizar la matrícula, el aspirante debe presentar:

- Documento de identidad.
- Formulario de inscripción diligenciado.
- Datos de contacto actualizados.
- Autorización del acudiente en caso de menores de edad, si aplica.

## 4. Confirmación de cupo

El agente TodoSystem EduBot puede orientar sobre los cursos disponibles, pero no puede garantizar cupos. La disponibilidad debe ser confirmada directamente con el área administrativa del instituto.

## 5. Información sobre pagos

Los valores de matrícula, mensualidades o pagos asociados a cada curso deben ser confirmados con el área administrativa. La información puede variar según el curso, la jornada, la disponibilidad y las políticas vigentes.

## 6. Recomendación final

Antes de iniciar un curso, el estudiante debe confirmar:

- Nombre del curso.
- Duración.
- Horario.
- Modalidad.
- Requisitos.
- Valor actualizado.
- Fecha de inicio.
- Condiciones de certificación.

## 7. Fuente

Documento interno de referencia creado para el proyecto académico TodoSystem EduBot.
