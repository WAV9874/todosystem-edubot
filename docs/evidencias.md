# Evidencias del proyecto - TodoSystem EduBot

Este documento registra las evidencias funcionales y técnicas del proyecto **TodoSystem EduBot**, desarrollado para el Challenge Alura Agente - ONE AI For Tech.

El objetivo de este archivo es organizar las pruebas visuales, capturas de pantalla, registros de ejecución y evidencias necesarias para demostrar que el agente funciona correctamente.

---

## 1. Descripción general de las evidencias

Las evidencias del proyecto permiten demostrar que:

- El agente fue desarrollado y probado localmente.
- La base de conocimiento fue construida con archivos PDF y CSV.
- El sistema responde preguntas en lenguaje natural.
- El agente muestra fuentes consultadas.
- El proyecto cuenta con documentación técnica.
- El sistema registra la ejecución de las consultas.
- La aplicación será desplegada en Oracle Cloud Infrastructure.
